我怎么理解 AI Agent 应用开发

Agent 不是单纯调用模型，而是围绕目标把模型、上下文、工具、知识和工作流接起来。

从 Java 后端视角看 RAG

RAG 在工程里要考虑的不只是“能召回”，还包括切分粒度、上下文长度、异常兜底和结果可控性。

我做项目时最重视什么

功能闭环、接口清晰、可演示、可调试。先把链路跑通，再逐步优化精度和体验。
